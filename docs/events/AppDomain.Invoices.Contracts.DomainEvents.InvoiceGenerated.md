# InvoiceGenerated

- **Status:** Active
- **Version:** v1
- **Entity:** `invoice`
- **Type:** Domain Event
- **Topic:** `{env}.appdomain.internal.invoices.v1`
- **Estimated Payload Size:** 310 bytes ⚠️ *Contains dynamic properties*

## Description

Domain event published when an invoice is generated within the AppDomain system.
This is an internal domain event that triggers subsequent processing within the domain.

## When It's Triggered

This domain event is published when:
- An invoice is programmatically generated by the system
- Automated app_domain processes create new invoices
- Recurring invoice generation occurs

## Domain Usage

This domain event is handled within the same domain to:
- Trigger additional invoice processing workflows
- Apply business rules specific to generated invoices
- Initialize invoice-specific processes
